# AgentHub - 运维与审计

---

## 1. 业务概述

运维模块为 admin 提供全局视角：操作审计日志、实例运行日志、系统监控仪表盘、异步任务管理。

---

## 2. 审计日志

### 2.1 记录策略

所有关键操作自动写入 `sys_operation_log` 表：

| 操作 | operator_type | target_type | target_id | detail |
|------|---------------|-------------|-----------|--------|
| 用户注册 | user | user | new_user_id | `{"username": "alice"}` |
| 登录成功 | user | auth | user_id | `{"ip": "1.2.3.4"}` |
| 登录失败 | anonymous | auth | — | `{"username": "alice", "ip": "1.2.3.4"}` |
| 创建实例 | user | instance | instance_id | `{"name": "我的助手", "port": 9001}` |
| 启动实例 | user | instance | instance_id | `{}` |
| 停止实例 | user | instance | instance_id | `{}` |
| 删除实例 | user | instance | instance_id | `{"container_name": "hermes-alice-001"}` |
| 创建密钥 | user | secret | secret_id | `{"name": "DeepSeek 主密钥"}` |
| 修改渠道 | user | channel | config_id | `{"platform": "weixin"}` |
| 用户管理 | admin | user | target_user_id | `{"action": "change_role"}` |

### 2.2 日志写入实现

```python
# 装饰器方式记录审计日志
async def audit_log(
    db: AsyncSession,
    user_id: int | None,
    action: str,
    target_type: str,
    target_id: int | None,
    detail: dict = None,
    ip: str = None
):
    log = SysOperationLog(
        user_id=user_id,
        action=action,
        target_type=target_type,
        target_id=target_id,
        detail=json.dumps(detail or {}, ensure_ascii=False),
        ip_address=ip,
    )
    db.add(log)
    await db.commit()
```

## 3. 运行日志

Hermes 容器的运行日志通过 docker-py 获取，并可选择性地持久化到 `instance_runtime_log` 表。

### 3.1 实时日志流

通过 WebSocket 推送:

```
前端 → ws://host:8000/ws/logs/{instance_id}
后端 → container.logs(stream=True, follow=True)
后端 → 逐行推送到 WebSocket
```

### 3.2 日志解析

```python
def parse_log_line(line: str) -> dict:
    """解析 Hermes 容器日志行"""
    # 格式: 2026-05-08T10:00:00 [INFO] Gateway started
    match = re.match(r'(\d{4}-\d{2}-\d{2}T[\d:]+)\s+\[(\w+)\]\s+(.*)', line)
    if match:
        return {"time": match.group(1), "level": match.group(2).lower(), "message": match.group(3)}
    return {"time": None, "level": "info", "message": line}
```

## 5. 异步任务框架

### 5.1 任务场景

| 任务类型 | 场景 | 超时 |
|----------|------|------|
| instance_create | 拉取镜像 + 创建容器 | 300s |
| instance_delete | 停止 + 删除容器 + 清理数据 | 60s |
| batch_restart | 批量重启所有实例 | 600s |

### 5.2 内存队列实现

MVP 阶段不使用 Redis/Celery，用 Python 内存队列 + asyncio:

```python
import asyncio
from collections import deque

class TaskQueue:
    def __init__(self, max_workers: int = 3):
        self.queue: deque = deque()
        self.semaphore = asyncio.Semaphore(max_workers)

    async def submit(self, task_id: str, coro):
        self.queue.append(task_id)
        asyncio.create_task(self._run(task_id, coro))

    async def _run(self, task_id: str, coro):
        async with self.semaphore:
            try:
                await update_task_status(task_id, "running")
                result = await coro
                await update_task_status(task_id, "completed", result=result)
            except Exception as e:
                await update_task_status(task_id, "failed", error=str(e))
            finally:
                self.queue.remove(task_id)

# 全局单例
task_queue = TaskQueue(max_workers=3)
```
