# AgentHub - 渠道管理

---

## 1. 业务概述

Hermes Agent 原生支持多个消息渠道（微信公众号、Telegram、Discord、Slack 等）。AgentHub 将渠道配置集中管理，让用户通过 Web 界面为每个实例配置和启用不同的消息渠道。

渠道的实际消息处理由 Hermes 容器完成，AgentHub 只负责：
1. 管理渠道配置（存储在 `instance_channel_config` 表）
2. 将配置注入容器环境变量
3. 监控渠道连接状态

## 3. 渠道配置生命周期

```
用户填写渠道配置
    │
    ▼
配置保存到 instance_channel_config 表
    │
    ▼
生成环境变量 dict（合并到 env_config）
    │
    ▼
启动/重启实例时注入容器
    │
    ▼
Hermes 容器根据环境变量自动连接对应平台
    │
    ▼
AgentHub 定期检测渠道连接状态
```

> 渠道配置变更后必须重启实例才能生效。前端在保存配置时给出提示。

## 4. API 接口定义

### 4.1 GET /api/channel-templates

获取所有支持的渠道模板（用于前端渲染动态表单）。

**成功响应**:

```json
{
  "code": 0,
  "data": [
    {
      "platform": "weixin",
      "name": "微信公众号",
      "fields": [
        {"key": "WEIXIN_DM_POLICY", "label": "私信策略", "type": "select",
         "options": ["open", "keyword", "admin_only"], "default": "open"},
        {"key": "WEIXIN_APP_ID", "label": "AppID", "type": "text", "required": true},
        {"key": "WEIXIN_APP_SECRET", "label": "AppSecret", "type": "password", "required": true},
        {"key": "WEIXIN_TOKEN", "label": "Token", "type": "text", "required": true}
      ]
    }
  ]
}
```

### 4.2 GET /api/instances/{id}/channels

获取实例已配置的渠道列表。

**成功响应**:

```json
{
  "code": 0,
  "data": [
    {
      "id": 1,
      "platform": "weixin",
      "platform_name": "微信公众号",
      "status": "connected",
      "config": {
        "WEIXIN_DM_POLICY": "open",
        "WEIXIN_APP_ID": "wx****1234",
        "WEIXIN_APP_SECRET": "****",
        "WEIXIN_TOKEN": "****"
      },
      "updated_at": "2026-05-08T10:00:00"
    }
  ]
}
```

> 敏感字段返回时脱敏。

### 4.3 PUT /api/instances/{id}/channels/{platform}

创建或更新实例的渠道配置。

**请求**:

```json
{
  "config": {
    "WEIXIN_DM_POLICY": "open",
    "WEIXIN_APP_ID": "wx123456",
    "WEIXIN_APP_SECRET": "secret123",
    "WEIXIN_TOKEN": "mytoken"
  }
}
```

**校验**: 根据 channel template 的 `required` 字段校验必填项。

**成功响应**:

```json
{
  "code": 0,
  "data": {
    "platform": "weixin",
    "status": "pending_restart",
    "message": "配置已保存，请重启实例使配置生效"
  }
}
```

### 4.4 DELETE /api/instances/{id}/channels/{platform}

删除实例的渠道配置。

**成功响应**:

```json
{
  "code": 0,
  "data": { "message": "渠道配置已删除，请重启实例使配置生效" }
}
```

**前端动态表单渲染**:

```typescript
// 根据 channel template 动态生成表单
function renderField(field: ChannelField) {
  switch (field.type) {
    case 'text':     return ElInput
    case 'password': return ElInput({ showPassword: true })
    case 'select':   return ElSelect({ options: field.options })
  }
}
```
