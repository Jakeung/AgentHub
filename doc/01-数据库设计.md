# AgentHub - 数据库设计

> 数据库: SQLite 3  
> ORM: SQLAlchemy 2.0 (async)  
> 文件位置: `data/agenthub.db`

---

## 2. 完整 DDL（可直接执行）

```sql
-- ============================================================
-- AgentHub 数据库 DDL
-- 数据库: SQLite 3
-- 执行: sqlite3 data/agenthub.db < this_file.sql
-- ============================================================

PRAGMA journal_mode=WAL;
PRAGMA foreign_keys=ON;

-- ============================================================
-- 1. 角色表
-- ============================================================
CREATE TABLE IF NOT EXISTS sys_role (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    name        TEXT    NOT NULL UNIQUE,
    description TEXT,
    created_at  TEXT    NOT NULL DEFAULT (datetime('now'))
);

-- ============================================================
-- 2. 权限表
-- ============================================================
CREATE TABLE IF NOT EXISTS sys_permission (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    code        TEXT    NOT NULL UNIQUE,      -- 格式: module:action
    name        TEXT    NOT NULL,
    module      TEXT    NOT NULL,
    created_at  TEXT    NOT NULL DEFAULT (datetime('now'))
);

-- ============================================================
-- 3. 角色-权限关联表
-- ============================================================
CREATE TABLE IF NOT EXISTS sys_role_permission (
    id            INTEGER PRIMARY KEY AUTOINCREMENT,
    role_id       INTEGER NOT NULL REFERENCES sys_role(id) ON DELETE CASCADE,
    permission_id INTEGER NOT NULL REFERENCES sys_permission(id) ON DELETE CASCADE,
    UNIQUE(role_id, permission_id)
);

-- ============================================================
-- 4. 用户表
-- ============================================================
CREATE TABLE IF NOT EXISTS sys_user (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    username        TEXT    NOT NULL UNIQUE,
    email           TEXT    UNIQUE,
    password_hash   TEXT    NOT NULL,       -- bcrypt
    name            TEXT,                   -- 显示名
    role_id         INTEGER NOT NULL REFERENCES sys_role(id),
    is_active       INTEGER NOT NULL DEFAULT 1,
    last_login_at   TEXT,
    created_at      TEXT    NOT NULL DEFAULT (datetime('now')),
    updated_at      TEXT    NOT NULL DEFAULT (datetime('now'))
);

-- ============================================================
-- 5. Agent 实例表
-- ============================================================
CREATE TABLE IF NOT EXISTS agent_instance (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    owner_user_id   INTEGER NOT NULL REFERENCES sys_user(id),
    name            TEXT    NOT NULL,          -- 实例显示名
    container_name  TEXT    NOT NULL UNIQUE,   -- Docker 容器名
    port            INTEGER NOT NULL UNIQUE,   -- 宿主机映射端口
    status          TEXT    NOT NULL DEFAULT 'stopped',
                    -- 取值: creating/starting/running/stopping/stopped/restarting/error/deleted
    health_status   TEXT    NOT NULL DEFAULT 'unknown',
                    -- 取值: healthy/unhealthy/unknown
    cpu_limit       REAL    NOT NULL DEFAULT 1.0,    -- CPU 核数
    memory_limit_mb INTEGER NOT NULL DEFAULT 2048,   -- 内存 MB
    data_dir        TEXT    NOT NULL,                 -- 宿主机数据目录
    env_config      TEXT    NOT NULL DEFAULT '{}',    -- JSON: 环境变量配置
    error_message   TEXT,
    created_at      TEXT    NOT NULL DEFAULT (datetime('now')),
    updated_at      TEXT    NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_instance_owner_status ON agent_instance(owner_user_id, status);
CREATE INDEX idx_instance_updated ON agent_instance(updated_at DESC);

-- ============================================================
-- 6. 实例模型配置表
-- ============================================================
CREATE TABLE IF NOT EXISTS instance_model_config (
    id            INTEGER PRIMARY KEY AUTOINCREMENT,
    instance_id   INTEGER NOT NULL REFERENCES agent_instance(id) ON DELETE CASCADE,
    provider      TEXT    NOT NULL DEFAULT 'deepseek',
                  -- 取值: deepseek/openai/claude/custom
    model_name    TEXT    NOT NULL DEFAULT 'deepseek-chat',
    api_secret_id INTEGER REFERENCES api_secret(id),
    extra_config  TEXT    NOT NULL DEFAULT '{}',  -- JSON: 温度/max_tokens 等
    created_at    TEXT    NOT NULL DEFAULT (datetime('now')),
    updated_at    TEXT    NOT NULL DEFAULT (datetime('now')),
    UNIQUE(instance_id)
);

-- ============================================================
-- 7. API 密钥表
-- ============================================================
CREATE TABLE IF NOT EXISTS api_secret (
    id               INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id          INTEGER NOT NULL REFERENCES sys_user(id),
    provider         TEXT    NOT NULL,             -- deepseek/openai/claude/custom
    secret_ciphertext TEXT   NOT NULL,             -- AES-256-GCM 密文
    secret_mask      TEXT    NOT NULL,             -- 掩码显示: sk-****xxxx
    name             TEXT,                         -- 用户给密钥起的别名
    is_default       INTEGER NOT NULL DEFAULT 0,
    status           TEXT    NOT NULL DEFAULT 'active',  -- active/disabled
    last_verified_at TEXT,
    created_at       TEXT    NOT NULL DEFAULT (datetime('now')),
    updated_at       TEXT    NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_secret_user ON api_secret(user_id, provider);

-- ============================================================
-- 8. 实例渠道配置表
-- ============================================================
CREATE TABLE IF NOT EXISTS instance_channel_config (
    id            INTEGER PRIMARY KEY AUTOINCREMENT,
    instance_id   INTEGER NOT NULL REFERENCES agent_instance(id) ON DELETE CASCADE,
    platform      TEXT    NOT NULL,
                  -- 取值: weixin/telegram/discord/slack
    status        TEXT    NOT NULL DEFAULT 'disconnected',
                  -- 取值: connected/disconnected/error
    config_json   TEXT    NOT NULL DEFAULT '{}',    -- 平台特定配置（密文）
    connected_at  TEXT,
    error_message TEXT,
    created_at    TEXT    NOT NULL DEFAULT (datetime('now')),
    updated_at    TEXT    NOT NULL DEFAULT (datetime('now')),
    UNIQUE(instance_id, platform)
);

-- ============================================================
-- 9. 实例运行日志表
-- ============================================================
CREATE TABLE IF NOT EXISTS instance_runtime_log (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    instance_id INTEGER NOT NULL REFERENCES agent_instance(id) ON DELETE CASCADE,
    level       TEXT    NOT NULL DEFAULT 'info',   -- debug/info/warn/error
    message     TEXT    NOT NULL,
    source      TEXT,                              -- docker/gateway/channel
    created_at  TEXT    NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_runtime_log_instance ON instance_runtime_log(instance_id, created_at DESC);

-- ============================================================
-- 10. 系统操作审计日志表
-- ============================================================
CREATE TABLE IF NOT EXISTS sys_operation_log (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id     INTEGER NOT NULL,
    module      TEXT    NOT NULL,    -- auth/instance/secret/channel/admin
    action      TEXT    NOT NULL,    -- create/update/delete/start/stop/login/login_failed
    resource_id TEXT,                -- 操作目标 ID
    detail      TEXT,                -- JSON 详情
    ip          TEXT,
    created_at  TEXT    NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_audit_user_time ON sys_operation_log(user_id, created_at DESC);
CREATE INDEX idx_audit_module_time ON sys_operation_log(module, created_at DESC);

-- ============================================================
-- 11. 异步任务表
-- ============================================================
CREATE TABLE IF NOT EXISTS async_task (
    id            INTEGER PRIMARY KEY AUTOINCREMENT,
    task_type     TEXT    NOT NULL,    -- instance.start/instance.stop/health.check 等
    target_id     TEXT,                -- 操作目标（instance_id 等）
    status        TEXT    NOT NULL DEFAULT 'pending',
                  -- 取值: pending/running/success/failed/dead
    payload       TEXT    NOT NULL DEFAULT '{}',   -- JSON
    result        TEXT,                             -- JSON
    retry_count   INTEGER NOT NULL DEFAULT 0,
    max_retries   INTEGER NOT NULL DEFAULT 5,
    next_retry_at TEXT,
    created_by    INTEGER,
    created_at    TEXT    NOT NULL DEFAULT (datetime('now')),
    updated_at    TEXT    NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_task_status ON async_task(status, next_retry_at);
```

## 4. 表说明速查

| 表名 | 用途 | 关键关系 |
|------|------|----------|
| `sys_role` | 角色定义 | — |
| `sys_permission` | 权限码定义 | — |
| `sys_role_permission` | 角色↔权限 N:M | role_id, permission_id |
| `sys_user` | 用户账号 | role_id → sys_role |
| `agent_instance` | Hermes 容器实例 | owner_user_id → sys_user |
| `instance_model_config` | 实例的模型配置 | instance_id → agent_instance |
| `api_secret` | 用户的 API Key（密文） | user_id → sys_user |
| `instance_channel_config` | 实例的渠道配置 | instance_id → agent_instance |
| `instance_runtime_log` | 实例运行日志 | instance_id → agent_instance |
| `sys_operation_log` | 全局审计日志 | user_id（不用 FK，防级联） |
| `async_task` | 异步任务队列 | created_by（不用 FK） |

## 5. SQLite 注意事项

1. **WAL 模式**: `PRAGMA journal_mode=WAL;` 支持并发读写。
2. **外键**: 必须 `PRAGMA foreign_keys=ON;` 每次连接都要执行。
3. **日期**: SQLite 无原生日期类型，统一用 ISO 8601 TEXT（`datetime('now')`）。
4. **JSON**: SQLite 3.38+ 支持 `json()` 函数，`env_config` 等字段存 JSON TEXT。
5. **备份**: `cp data/agenthub.db data/agenthub.db.bak` 即可。
6. **并发限制**: 写操作串行化，高并发写场景需升级 PostgreSQL（当前规模无需）。
